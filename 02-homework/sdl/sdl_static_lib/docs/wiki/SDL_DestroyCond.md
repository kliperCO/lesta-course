# SDL_DestroyCond

Destroy a condition variable.

## Syntax

```c
void SDL_DestroyCond(SDL_cond *cond);

```

## Function Parameters

|              |                                   |
| ------------ | --------------------------------- |
| **cond**     | the condition variable to destroy |

## Version

This function is available since SDL 3.0.0.
